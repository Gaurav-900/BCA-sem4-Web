Unit I

Introduction to PHP

PHP (Hypertext Preprocessor) is a widely-used, open-source, server-side scripting language designed for web development. This unit covers PHP's origin, purpose, working model, and how it integrates with web servers.

🎯 Learning Objectives

  • Understand what PHP is and its history
  • Know PHP's role in the web technology stack
  • Distinguish PHP from client-side scripting
  • Understand the client-server model with PHP

What is PHP?

Definition: PHP (Hypertext Preprocessor) is an open-source, general-purpose scripting language that is especially suited for web development and can be embedded directly into HTML. PHP code is executed on the server side, and the result is sent to the client's browser as plain HTML.

PHP was originally created by Rasmus Lerdorf in 1994. The name initially stood for "Personal Home Page Tools." Over time it evolved into a full-featured language and was renamed to the recursive acronym PHP: Hypertext Preprocessor.

How PHP Works — The Client-Server Model

1
Client sends HTTP request — browser requests a .php file
2
Web server intercepts — passes the file to the PHP interpreter
3
PHP engine executes — processes PHP code, may query database
4
HTML is generated — PHP produces pure HTML output
5
Response sent — browser receives and renders HTML

Embedding PHP in HTML

One of PHP's most powerful features is that it can be embedded directly within HTML. The PHP engine processes only the code within PHP tags and passes everything else through unchanged.

Embedding Patterns

<!DOCTYPE html>
<html>
<head>
  <title><?php echo "Dynamic Title"; ?></title>
</head>
<body>
  <h1>Welcome, <?= $username ?>!</h1>

  <?php if ($loggedIn): ?>
    <p>You are logged in.</p>
  <?php else: ?>
    <p>Please log in.</p>
  <?php endif; ?>

  <ul>
    <?php foreach ($items as $item): ?>
      <li><?= htmlspecialchars($item) ?></li>
    <?php endforeach; ?>
  </ul>
</body>
</html>
💡 Best Practice: Use htmlspecialchars() when outputting user data to prevent Cross-Site Scripting (XSS) attacks.

Viewing PHP in Browser

To view a PHP file in the browser:

  1. Save the file with .php extension inside the web server root (e.g., htdocs/ in XAMPP)
  2. Start Apache (and MySQL if needed)
  3. Open browser and navigate to http://localhost/yourfile.php
  4. Never open the file directly from the filesystem — PHP won't execute
⚠️ Common Mistake: Opening a .php file by double-clicking it will show raw PHP code, NOT the output. Always access PHP through a web server via localhost.

PHP Variables — Static & Global

Variable: A named memory location that holds a value. In PHP, all variables start with a $ sign followed by the variable name. PHP is case-sensitive for variable names.

Variable Rules

  • Must start with $ followed by a letter or underscore
  • Can contain letters, numbers, underscores
  • Case-sensitive: $name ≠ $Name
  • No type declaration needed

Variable Scope

ScopeDescriptionAccessible Where
LocalDeclared inside a functionOnly within that function
GlobalDeclared outside functionsAnywhere outside functions; inside functions only with global keyword
StaticDeclared with static inside functionRetains value between function calls
 
<?php
$globalVar = 100;  // Global variable

function testScope() {
    // Without global keyword, $globalVar is undefined here
    echo isset($globalVar) ? "exists" : "not found"; // "not found"
    
    // Access global variable using 'global' keyword
    global $globalVar;
    echo $globalVar; // 100
    
    // $GLOBALS superglobal (alternative)
    echo $GLOBALS['globalVar']; // 100
}

// ---- STATIC VARIABLE ----
function counter() {
    static $count = 0; // Initialized only ONCE
    $count++;
    echo $count . " ";
}

counter(); // 1
counter(); // 2
counter(); // 3
// Without static, each call would print "1"
?>
Static Variable: A local variable that persists its value between function calls. Initialized only once — the first time the function is called. Useful for counters, caching.

Control Structures — Conditional Statements

if…else Statement

<?php
$marks = 75;

// Simple if
if ($marks >= 40) {
    echo "Pass";
}

// if-else
if ($marks >= 60) {
    echo "Distinction";
} else {
    echo "Pass";
}

// if-elseif-else
if ($marks >= 75) {
    echo "First Division";
} elseif ($marks >= 60) {
    echo "Second Division";
} elseif ($marks >= 40) {
    echo "Third Division";
} else {
    echo "Fail";
}
?>

Switch Statement

<?php
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Start of work week";
        break;          // break prevents fall-through
    case "Friday":
        echo "End of work week";
        break;
    case "Saturday":
    case "Sunday":      // Multiple cases with same action
        echo "Weekend!";
        break;
    default:
        echo "Midweek";
}

// Switch uses loose comparison (==), not strict (===)
?>

Ternary Operator ( ? : )

<?php
// Syntax: condition ? value_if_true : value_if_false
$age = 20;
$status = ($age >= 18) ? "Adult" : "Minor";
echo $status; // Adult

// Null Coalescing (PHP 7+) — shorter version for null checks
$username = $_GET['user'] ?? "Guest";
// Equivalent to: isset($_GET['user']) ? $_GET['user'] : "Guest"

// Null coalescing assignment (PHP 7.4+)
$data['key'] ??= "default";
?>

Loops in PHP

<?php
// ---- WHILE LOOP ----
$i = 1;
while ($i <= 5) {
    echo $i . " ";
    $i++;
}
// Output: 1 2 3 4 5

// ---- DO-WHILE LOOP ----
$i = 10;
do {
    echo $i . " ";  // Executes at least once
    $i++;
} while ($i <= 5);
// Output: 10 (condition fails, but body executed once)

// ---- FOR LOOP ----
for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";
}
// Syntax: for(init; condition; increment)

// ---- FOREACH LOOP ----
$fruits = ["apple", "banana", "cherry"];
foreach ($fruits as $fruit) {
    echo $fruit . " ";
}
// With key => value
$ages = ["Alice" => 25, "Bob" => 30];
foreach ($ages as $name => $age) {
    echo "$name is $age years old. ";
}

// ---- BREAK & CONTINUE ----
for ($i = 1; $i <= 10; $i++) {
    if ($i == 6) break;      // Exits loop entirely
    if ($i % 2 == 0) continue; // Skips even numbers
    echo $i . " ";
}
// Output: 1 3 5

// break with a number: break 2 breaks out of 2 nested loops

// ---- EXIT & DIE ----
// exit("Error message") — terminates entire script
// die("Error") — alias of exit
?>

exit, die, return

StatementPurposeEffect
exit()Terminate script executionStops entire PHP script; can output message or exit code
die()Alias for exit()Same as exit; commonly used after DB connection failure
returnExit from current functionReturns control to caller; in global scope, behaves like exit

Arrays in PHP

Array: An ordered map that associates keys with values. PHP arrays can hold values of different types, including other arrays. Arrays in PHP are implemented as an ordered hash map.

Types of Arrays

<?php
// =============================================
// 1. NUMERIC (INDEXED) ARRAY
// =============================================
$colors = ["Red", "Green", "Blue"];  // Modern syntax
$nums = array(10, 20, 30);           // Old syntax

echo $colors[0];   // Red
echo $colors[2];   // Blue

// Useful functions
echo count($colors);   // 3
echo implode(", ", $colors);  // Red, Green, Blue
sort($colors);   // Sort ascending

// =============================================
// 2. ASSOCIATIVE ARRAY
// =============================================
$student = [
    "name"    => "Rahul",
    "age"     => 21,
    "branch"  => "BCA"
];

echo $student["name"];    // Rahul
echo $student["branch"];  // BCA

// Loop through
foreach ($student as $key => $value) {
    echo "$key: $value\n";
}

// Functions
ksort($student);   // Sort by key
asort($student);   // Sort by value, maintain key association
array_keys($student);   // ["name", "age", "branch"]
array_values($student); // ["Rahul", 21, "BCA"]

// =============================================
// 3. MULTIDIMENSIONAL ARRAY
// =============================================
$students = [
    ["name" => "Alice", "marks" => 85],
    ["name" => "Bob",   "marks" => 72],
    ["name" => "Carol", "marks" => 91]
];

// Access: $students[row][column]
echo $students[0]["name"];   // Alice
echo $students[2]["marks"];  // 91

// Loop
foreach ($students as $s) {
    echo $s["name"] . ": " . $s["marks"] . "\n";
}
?>

Unit II — Exam Questions & Answers

2-Mark Questions

Q1. What is the difference between == and === in PHP?
== (loose equality) compares values after type conversion. === (strict equality) compares both value AND type without conversion. Example: "1" == 1 is true, but "1" === 1 is false.
Q2. What is a static variable?
A static variable is declared with the static keyword inside a function. It retains its value between function calls instead of being re-initialized. Initialized only once.
Q3. Difference between break and continue?
break exits the loop entirely. continue skips the rest of the current iteration and moves to the next one.

5-Mark Questions

Q4. Explain types of arrays in PHP with examples.
1. Numeric/Indexed Array: Keys are automatically assigned integers starting at 0. $a = ["Red", "Green", "Blue"]; — Access: $a[0] = "Red"

2. Associative Array: Keys are user-defined strings. $s = ["name"=>"Alice", "age"=>20]; — Access: $s["name"]

3. Multidimensional Array: Array of arrays. $data = [["name"=>"A","marks"=>80], ["name"=>"B","marks"=>70]]; — Access: $data[0]["name"]
Q5. Explain all PHP loop types with syntax.
while: Checks condition first; may execute 0 times.
do-while: Executes at least once; checks condition after.
for: Count-controlled; init, condition, and increment in one line.
foreach: For iterating over arrays; no index management needed.

10-Mark Questions

Q6. Explain all PHP data types with examples. What is type juggling?
PHP has 8 primitive types:
Scalar: int, float, bool, string — Compound: array, object — Special: null, resource

Type Juggling: PHP automatically converts types during operations. Example: "5" + 3 = 8 (PHP converts string "5" to int 5).
This can cause bugs — use strict comparison === when type matters.

Type Casting: Manual conversion using (int), (float), (string), (bool), (array) casts.

Key Points for Exam

  • Variable names are case-sensitive and start with $
  • static variables retain value between calls; initialized once
  • global keyword needed to access global vars inside functions
  • do-while executes body at least once
  • foreach is designed for arrays: foreach($arr as $k => $v)
  • switch uses loose (==) comparison
  • Ternary: condition ? true_value : false_value
  • Null coalescing: $x ?? "default"
  • Array functions: sort, ksort, asort, array_push, array_pop, in_array, count

Searching & Replacing

<?php
$str = "Hello PHP World PHP";

// Search
echo strpos($str, "PHP");      // 6 (first occurrence)
echo strrpos($str, "PHP");     // 16 (last occurrence)
echo strstr($str, "PHP");      // "PHP World PHP" (from first match)
echo stripos($str, "php");     // 6 (case-insensitive)
echo substr($str, 6, 3);       // "PHP" (start, length)
echo substr_count($str, "PHP"); // 2

<?php
// ---- BASIC EXCEPTION HANDLING ----

    $denominator = 0;
    if ($denominator == 0) {
        throw new Exception("Division by zero is not allowed!");
    }

    echo "Error caught: " . $e->getMessage();
    echo "\nLine: " . $e->getLine();
    echo "\nFile: " . $e->getFile();
} finally {
    echo "\nThis block always executes.";
    // Use for cleanup: close DB connections, files, etc.
}

function riskyOperation($type) {
    if ($type == "invalid") {
        throw new InvalidArgumentException("Invalid argument!");
    }
    if ($type == "runtime") {
        throw new RuntimeException("Runtime error!");
    }
}

try {
    riskyOperation("invalid");
} catch (InvalidArgumentException $e) {
    echo "Invalid argument: " . $e->getMessage();
} catch (RuntimeException $e) {
    echo "Runtime error: " . $e->getMessage();
} catch (Exception $e) {
    echo "General exception: " . $e->getMessage();
}

// ---- CUSTOM EXCEPTION CLASS ----
class DatabaseException extends Exception {
    private $query;
    
    public function __construct($message, $query = "", $code = 0) {
        parent::__construct($message, $code);
        $this->query = $query;
    }
    
    public function getQuery() {
        return $this->query;
    }
    
    public function __toString() {
        return "DatabaseException [{$this->code}]: {$this->message}\nQuery: {$this->query}";
    }
}

try {
    throw new DatabaseException("Query failed", "SELECT * FROM users", 500);
} catch (DatabaseException $e) {
    echo $e->getQuery();
}

<?php
// =============================================
// OPENING AND CLOSING FILES
// =============================================
$filename = "data.txt";

// Open file
$handle = fopen($filename, "w") or die("Cannot open file!");

// Write to file
fwrite($handle, "Hello, PHP File Handling!\n");
fwrite($handle, "Second line of text.\n");

// Close file — always close after use!
fclose($handle);

// =============================================
// READING FILES
// =============================================
$handle = fopen($filename, "r");

// Read entire file at once
$content = fread($handle, filesize($filename));
echo $content;
fclose($handle);

    $line = fgets($handle);        // Read one line

}
fclose($handle);

// Append: use FILE_APPEND flag
file_put_contents("output.txt", "\nAppended text", FILE_APPEND);

// Read file into array (line by line)
$lines = file("data.txt", FILE_IGNORE_NEW_LINES | FILE_SKIP_EMPTY_LINES);
foreach ($lines as $line) {

}

copy("data.txt", "backup.txt");       // Copy file

echo filemtime("data.txt");       // Last modified timestamp
echo is_readable("data.txt");
echo is_writable("data.txt");

echo basename("path/to/file.php"); // "file.php"

Database Handling — MySQL & PHP

PHP can connect to MySQL databases using MySQLi (MySQL Improved) or PDO (PHP Data Objects). MySQLi is MySQL-specific; PDO is database-agnostic.

<?php
// =============================================
// 1. CONNECTION WITH MySQLi
// =============================================
$host     = "localhost";
$username = "root";
$password = "";
$database = "college_db";

// Object-oriented style
$conn = new mysqli($host, $username, $password, $database);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully!";

// Procedural style (alternative)
$conn = mysqli_connect($host, $username, $password, $database);
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}

// =============================================
// 2. CREATE TABLE
// =============================================
$sql = "CREATE TABLE IF NOT EXISTS students (
    id       INT AUTO_INCREMENT PRIMARY KEY,
    name     VARCHAR(100) NOT NULL,
    email    VARCHAR(150) UNIQUE,
    branch   VARCHAR(50),
    marks    FLOAT,
    created  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)";
$conn->query($sql);

// =============================================
// 3. INSERT (CREATE)
// =============================================
$name   = "Alice";
$email  = "alice@example.com";
$branch = "BCA";
$marks  = 85.5;

$sql = "INSERT INTO students (name, email, branch, marks)
        VALUES ('$name', '$email', '$branch', $marks)";

if ($conn->query($sql) === TRUE) {
    echo "Record inserted! ID: " . $conn->insert_id;
}

// =============================================
// 4. SELECT (READ)
// =============================================
// Basic SELECT
$result = $conn->query("SELECT * FROM students");
while ($row = $result->fetch_assoc()) {
    echo $row['name'] . " - " . $row['marks'] . "\n";
}

// With ORDER BY and ALIAS
$sql = "SELECT name, marks, branch AS dept 
        FROM students 
        WHERE marks > 60 
        ORDER BY marks DESC";
$result = $conn->query($sql);
$count  = $result->num_rows;
echo "Found $count students";

// =============================================
// 5. UPDATE
// =============================================
$sql = "UPDATE students SET marks = 90 WHERE name = 'Alice'";
if ($conn->query($sql)) {
    echo "Updated rows: " . $conn->affected_rows;
}

// =============================================
// 6. DELETE
// =============================================
$sql = "DELETE FROM students WHERE id = 5";
$conn->query($sql);

// =============================================
// 7. TRUNCATE (Remove all rows, keep structure)
// =============================================
$conn->query("TRUNCATE TABLE students");

// =============================================
// 8. PREPARED STATEMENTS (Prevent SQL Injection)
// =============================================
$stmt = $conn->prepare("INSERT INTO students (name, email, marks) VALUES (?, ?, ?)");
$stmt->bind_param("ssd", $name, $email, $marks);  // s=string, d=double
$name  = "Bob";
$email = "bob@example.com";
$marks = 78.0;
$stmt->execute();
echo "Inserted ID: " . $stmt->insert_id;
$stmt->close();

// Prepared SELECT
$stmt = $conn->prepare("SELECT * FROM students WHERE branch = ?");
$stmt->bind_param("s", $branch);
$branch = "BCA";
$stmt->execute();
$result = $stmt->get_result();
while ($row = $result->fetch_assoc()) {
    echo $row['name'] . "\n";
}
$stmt->close();

// =============================================
// 9. CLOSE CONNECTION
// =============================================
$conn->close();

// =============================================
// 10. PDO CONNECTION (Database-agnostic)
// =============================================
try {
    $pdo = new PDO("mysql:host=localhost;dbname=college_db", "root", "");
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    
    // PDO prepared statement
    $stmt = $pdo->prepare("SELECT * FROM students WHERE marks > :minMarks");
    $stmt->execute(['minMarks' => 70]);
    $students = $stmt->fetchAll(PDO::FETCH_ASSOC);
    
    foreach ($students as $s) {
        echo $s['name'] . "\n";
    }
} catch (PDOException $e) {
    echo "PDO Error: " . $e->getMessage();
}
?>

📌 Database Connectivity Pattern

$conn = new mysqli("localhost","root","","db");
if ($conn->connect_error) die($conn->connect_error);

// CRUD
$conn->query("SELECT/INSERT/UPDATE/DELETE ...");
$result = $conn->query("SELECT * FROM t");
while ($row = $result->fetch_assoc()) { }

// Prepared (safe)
$stmt = $conn->prepare("INSERT INTO t (a,b) VALUES (?,?)");
$stmt->bind_param("si", $strVal, $intVal);
$stmt->execute();
$conn->close();

// Simple write

file_put_contents("file.txt", $data, FILE_APPEND);

// Full control
$h = fopen("file.txt", "r");
while (!feof($h)) { $line = fgets($h); }
fclose($h);

// Management

rename($old, $new); // Rename/move

📌 Error Handling Pattern

    if (errorCondition) {
        throw new Exception("Message", $code);

    // normal code

    echo $e->getMessage();
} catch (Exception $e) {
    echo $e->getMessage();
    // $e->getCode(), getFile(), getLine(), getTrace()
} finally {
    // always runs — cleanup
}
⚡ Exam Prep

Final Exam Crash Revision

This is your last-minute revision guide. Read through all points once before your exam.

🔑 Unit I — Must Know

  • PHP = server-side; browser gets HTML only
  • Created by Rasmus Lerdorf, 1994
  • Opening tag: <?php … ?>
  • Standard tag always works; short tag needs config
  • XAMPP: Apache + MySQL + PHP + Perl
  • Config file = php.ini
  • Must serve via web server, not file://
  • phpinfo() = view all PHP settings

🔑 Unit II — Must Know

  • PHP = loosely typed (dynamic typing)
  • 8 types: int, float, bool, string, array, object, null, resource
  • === checks value AND type (strict)
  • static var: retains value between calls
  • global keyword: access global var in function
  • do-while: always executes body at least once
  • foreach: foreach($arr as $k => $v)
  • break=exit loop; continue=skip iteration
  • Ternary: cond ? val1 : val2
  • Null coalesce: $a ?? "default"

🔑 Unit III — Must Know

  • Double quotes interpolate variables; single don't
  • Heredoc: interpolates; Nowdoc: doesn't
  • strpos: first occurrence; strrpos: last
  • explode=string→array; implode=array→string
  • PHP regex uses PCRE via preg_* functions
  • By reference: &$var modifies original
  • $_GET: URL; $_POST: body; $_REQUEST: both+cookie
  • session_start() BEFORE any output
  • Cookies stored in browser; sessions on server
  • Use htmlspecialchars() to prevent XSS

🔑 Unit IV — Must Know

  • Exception: catchable; Error: usually fatal
  • try→throw→catch→finally
  • finally: always runs (for cleanup)
  • Custom exception: extend Exception class
  • File modes: r, w(truncate), a(append), x(create-only)
  • fopen, fread, fwrite, fclose
  • Simpler: file_get/put_contents
  • unlink=delete; rename=rename; copy=copy
  • MySQLi: new mysqli(host,user,pass,db)
  • Prepared statements prevent SQL injection
  • TRUNCATE: removes all rows, keeps structure
  • insert_id: last auto-increment ID

⚠️ Common Mistakes Students Make

  1. Opening PHP file as file:// instead of http://localhost — PHP won't execute
  2. Forgetting semicolons at end of PHP statements
  3. Using == instead of === when type matters
  4. Calling session_start() after HTML output — causes header error
  5. Not closing fopen() handles — causes file locks
  6. Embedding user input directly in SQL — SQL injection vulnerability
  7. Forgetting break in switch — causes fall-through to next case
  8. Using 'w' mode when you mean 'a' — 'w' truncates/erases file!
  9. Confusing strpos() returning 0 with false — use === false, not == false
  10. Forgetting & in pass-by-reference — function will NOT modify original
