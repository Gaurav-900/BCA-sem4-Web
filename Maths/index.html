Unit I · Topic 1

Sets

🎯 Learning Objectives

  • Define a set and represent it in different ways
  • Identify and classify types of sets
  • Perform operations on sets (union, intersection, complement, difference)
  • Draw and interpret Venn diagrams
  • Apply the Principle of Inclusion–Exclusion

1.1 Definition of a Set

Definition: A set is a well-defined collection of distinct objects, called elements or members. A set is usually denoted by capital letters A, B, C, … and its elements by lower-case letters.

Example: A = {1, 2, 3, 4, 5} ; B = {a, e, i, o, u} ; C = {x | x is a prime number less than 10} = {2, 3, 5, 7}

If element a belongs to set A, we write a ∈ A; if not, a ∉ A.

1.2 Representation of Sets

MethodDescriptionExample
Roster / TabularList all elements inside braces{1, 2, 3, 4}
Set-Builder / RuleDefine property of elements{x | x ∈ ℕ, x ≤ 4}
DescriptiveDescribe in words"Set of first four natural numbers"

1.3 Types of Sets

Empty / Null Set (∅)

A set with no elements. e.g., {x | x ∈ ℕ, x < 0} = ∅

Singleton Set

Exactly one element. e.g., {5}

Finite Set

Countable number of elements. e.g., {1,2,3}

Infinite Set

Elements continue without end. e.g., ℕ = {1,2,3,…}

Universal Set (U)

The all-encompassing set for a given context

Equal Sets

A = B iff every element of A is in B and vice versa

Equivalent Sets

Same cardinality (number of elements)

Disjoint Sets

A ∩ B = ∅ (no common elements)

1.4 Subsets & Power Set

Subset: A ⊆ B if every element of A is also in B. A ⊂ B (proper subset) if A ⊆ B and A ≠ B.
Power Set P(A): The set of ALL subsets of A. If |A| = n, then |P(A)| = 2n.

Example: A = {1, 2} → P(A) = {∅, {1}, {2}, {1,2}} → |P(A)| = 2² = 4

1.5 Operations on Sets

UnionA ∪ B = {x | x ∈ A or x ∈ B}
IntersectionA ∩ B = {x | x ∈ A and x ∈ B}
DifferenceA − B = {x | x ∈ A and x ∉ B}
ComplementA′ = Aᶜ = U − A = {x | x ∈ U, x ∉ A}
Sym. Diff.A △ B = (A−B) ∪ (B−A)

1.6 Venn Diagrams

Venn diagrams use overlapping circles inside a rectangle (universal set) to visualise set relationships.

1.7 Laws of Set Theory

Unit I · Topic 2

Relations

2.1 Cartesian Product

Definition: A × B = {(a, b) | a ∈ A, b ∈ B}. The Cartesian product is the set of all ordered pairs.

Example: A={1,2}, B={x,y} → A×B = {(1,x),(1,y),(2,x),(2,y)}

If |A|=m and |B|=n, then |A×B| = m×n.

2.2 Definition of Relation

Definition: A relation R from set A to set B is a subset of A×B. R ⊆ A×B. If (a,b) ∈ R, we write aRb.

The domain of R = {a ∈ A | ∃ b ∈ B, (a,b) ∈ R} and the range of R = {b ∈ B | ∃ a ∈ A, (a,b) ∈ R}.

2.3 Types of Relations

Reflexive

aRa for all a ∈ A
Every element is related to itself.

e.g., "is equal to" (a=a)

Symmetric

If aRb then bRa
Relation works both ways.

e.g., "is a sibling of"

Anti-Symmetric

If aRb and bRa then a=b
Equality is the only bidirectional pair.

e.g., "is less than or equal to"

Transitive

If aRb and bRc then aRc

e.g., "is an ancestor of"

Equivalence

Reflexive + Symmetric + Transitive

e.g., "is congruent to" (mod n)

Example: Let A = {1,2,3}, R = {(1,1),(2,2),(3,3),(1,2),(2,1)}.
✔ Reflexive (all (a,a) ∈ R)   ✔ Symmetric ((1,2)&(2,1) both present)   ✗ Transitive (check: (2,1) & (1,2) → (2,2) ✓, but (1,2)&(2,1)→(1,1)✓ — actually transitive here) → This R is an equivalence relation.
Unit I · Topic 3

Functions

3.1 Definition

Definition: A function f : A → B is a relation such that every element of A (domain) is associated with exactly one element of B (codomain). The set of actual outputs is the range.

3.2 Domain & Range

  • Domain: The set of all valid inputs (x values)
  • Codomain: The target set B
  • Range: {f(x) | x ∈ Domain} ⊆ Codomain
f(x) = √x  →  Domain = [0,∞), Range = [0,∞)

3.3 Types of Functions

TypeConditionExample
One-to-One (Injective)f(a)=f(b) ⟹ a=b. No two inputs give same output.f(x)=2x
Onto (Surjective)Every element of codomain has a preimage. Range = Codomain.f(x)=x³ on ℝ
BijectiveBoth injective AND surjective.f(x)=x+1
Many-to-OneTwo or more inputs map to same output.f(x)=x²
Constantf(x)=c for all xf(x)=5

3.4 Composite Functions

Composite Function (fog): (f∘g)(x) = f(g(x)). Apply g first, then f.
f(x) = x+2,  g(x) = 3x
(f∘g)(x) = f(g(x)) = f(3x) = 3x+2
(g∘f)(x) = g(f(x)) = g(x+2) = 3(x+2) = 3x+6
Note: f∘g ≠ g∘f (not commutative in general)

3.5 Inverse of a Function

Inverse f⁻¹: If f is bijective, then f⁻¹ : B → A such that f⁻¹(f(x)) = x. To find f⁻¹: swap x and y, then solve for y.
f(x) = 3x − 1
Step 1: y = 3x − 1
Step 2: x = 3y − 1 (swap)
Step 3: y = (x+1)/3
∴ f⁻¹(x) = (x+1)/3

Logic & Proofs

🎯 Learning Objectives

  • Apply logical connectives: AND, OR, NOT, implication, biconditional

1.1 Proposition

Proposition (Statement): A declarative sentence that is either TRUE (T) or FALSE (F), but not both.

Examples (Propositions)

  • "5 is a prime number" → T
  • "2+2=5" → F

Non-Propositions

  • "What time is it?" (question)
  • "Close the door." (command)

1.4 Tautology & Contradiction

Tautology

A compound proposition that is always TRUE, regardless of the truth values of its components.

Example: p ∨ ¬p (always T)

Contradiction (Fallacy)

A compound proposition that is always FALSE.

Example: p ∧ ¬p (always F)

Contingency

Neither tautology nor contradiction — truth value depends on components.

1.5 De Morgan's Laws (Logic)

Law 1¬(p ∧ q) ≡ ¬p ∨ ¬q
Law 2¬(p ∨ q) ≡ ¬p ∧ ¬q
Verify ¬(p ∧ q) ≡ ¬p ∨ ¬q: Construct truth table — both columns are identical, confirming the equivalence.
Unit II · Topic 2

Matrices

2.1 Definition

Matrix: A rectangular array of numbers arranged in rows and columns, enclosed in brackets. A matrix with m rows and n columns is called an m × n matrix. Element at row i, column j is denoted aij.

2.2 Types of Matrices

TypeDescriptionExample
Row MatrixSingle row (1×n)[1, 2, 3]
Column MatrixSingle column (n×1)[1; 2; 3]
Square MatrixRows = Columns (n×n)[[1,2],[3,4]]
Zero/Null MatrixAll elements = 0[[0,0],[0,0]]
Identity Matrix IDiagonal=1, rest=0[[1,0],[0,1]]
Diagonal MatrixNon-diagonal elements=0[[3,0],[0,5]]
Scalar MatrixDiagonal elements all equal[[k,0],[0,k]]
SymmetricA = Aᵀ (aᵢⱼ=aⱼᵢ)[[1,2],[2,3]]
Skew-SymmetricA = −Aᵀ (aᵢⱼ=−aⱼᵢ, diagonal=0)[[0,2],[−2,0]]

2.3 Matrix Operations

Addition & Subtraction

Only possible for matrices of the same order. Add/subtract corresponding elements.

(A+B)ij = aij + bij

Scalar Multiplication

Multiply every element by the scalar k.

(kA)ij = k·aij

Matrix Multiplication

Condition: A(m×p) × B(p×n) = C(m×n). Number of columns in A = Number of rows in B.
Cij = Σk=1 to p aik · bkj
Example: A = [[1,2],[3,4]], B = [[5,6],[7,8]]
C₁₁ = 1×5+2×7 = 5+14 = 19
C₁₂ = 1×6+2×8 = 6+16 = 22
C₂₁ = 3×5+4×7 = 15+28 = 43
C₂₂ = 3×6+4×8 = 18+32 = 50
AB = [[19,22],[43,50]]
Non-commutativity: In general, AB ≠ BA. Matrix multiplication is NOT commutative.

Transpose of a Matrix

Transpose Aᵀ: Obtained by interchanging rows and columns. (Aᵀ)ij = aji.
A = [[1,2,3],[4,5,6]] (2×3)
Aᵀ = [[1,4],[2,5],[3,6]] (3×2)
Properties of Transpose
(Aᵀ)ᵀ = A
(A+B)ᵀ = Aᵀ + Bᵀ
(AB)ᵀ = BᵀAᵀ
(kA)ᵀ = kAᵀ
Unit II · Topic 3

Determinants

3.1 Determinant of a Square Matrix

Definition: The determinant is a scalar value computed from a square matrix. Denoted det(A) or |A|.
1×1|A| = [a] → det = a
2×2|A| = |a b; c d| = ad − bc
3×3Expansion along Row 1:
|A| = a₁₁(a₂₂a₃₃−a₂₃a₃₂) − a₁₂(a₂₁a₃₃−a₂₃a₃₁) + a₁₃(a₂₁a₃₂−a₂₂a₃₁)
Example — 3×3:
A = [[2,1,3],[0,4,1],[5,2,0]]
|A| = 2(4×0−1×2) − 1(0×0−1×5) + 3(0×2−4×5)
= 2(0−2) − 1(0−5) + 3(0−20)
= 2(−2) − 1(−5) + 3(−20)
= −4 + 5 − 60 = −59

3.2 Minors & Cofactors

Minor Mij: The determinant of the submatrix obtained by deleting row i and column j.
Cofactor Cij: Cij = (−1)i+j Mij. The sign is determined by the checkerboard pattern:

3.3 Properties of Determinants

  1. det(Aᵀ) = det(A)
  2. If two rows (or columns) are identical, det(A) = 0
  3. Interchanging two rows changes the sign of the determinant
  4. Multiplying a row by k multiplies the determinant by k
  5. If all elements of a row are 0, det(A) = 0
  6. det(AB) = det(A) × det(B)
  7. Adding a multiple of one row to another does not change det(A)

3.4 Adjoint & Inverse of a Matrix

Adjoint (adj A): The transpose of the cofactor matrix of A.
Inverse A⁻¹: A⁻¹ = (1/det(A)) × adj(A), provided det(A) ≠ 0.
Example: A = [[1,2],[3,4]]
det(A) = 1×4−2×3 = 4−6 = −2
Cofactor matrix = [[4,−3],[−2,1]]
adj(A) = [[4,−2],[−3,1]] (transpose of cofactor matrix)
A⁻¹ = (1/−2)[[4,−2],[−3,1]] = [[−2,1],[1.5,−0.5]]

3.5 Area of a Triangle

FormulaArea = (1/2) |x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)|
Determinant FormArea = (1/2) |det([[x₁,y₁,1],[x₂,y₂,1],[x₃,y₃,1]])|
Vertices: (1,1),(2,3),(4,2)
Area = (1/2)|1(3−2)+2(2−1)+4(1−3)| = (1/2)|1+2−8| = (1/2)(5) = 2.5 sq. units

3.6 Cramer's Rule

Cramer's Rule: Solves a system of linear equations using determinants. For Ax = b:
x₁ = D₁/D,x₂ = D₂/D, x₃ = D₃/D
Where D=det(A), Dᵢ = det obtained by replacing column i with column b
Solve: 2x+y=5, x+3y=10
D = |2,1;1,3| = 6−1 = 5
D₁ = |5,1;10,3| = 15−10 = 5 → x = 5/5 = 1
D₂ = |2,5;1,10| = 20−5 = 15 → y = 15/5 = 3

Measures of Central Tendency

Central Tendency: A single value that represents the "centre" or "typical" value of a dataset. The three main measures are Mean, Median, and Mode.

3.1 Mean (Arithmetic Mean)

Ungroupedx̄ = (Σxᵢ)/n
Grouped (Direct)x̄ = Σ(f·x) / Σf
Assumed Meanx̄ = A + Σ(f·d)/Σf   where d = x−A
Step Deviationx̄ = A + (Σf·u/Σf)×h   where u=(x−A)/h
Ungrouped: Data: 10, 15, 20, 25, 30 → x̄ = (10+15+20+25+30)/5 = 100/5 = 20

Grouped:
Classfx (mid)f·x
10–2051575
20–30825200
30–40735245
Total20520
x̄ = 520/20 = 26

3.2 Median

Median: The middle value of an ordered dataset. It divides the distribution into two equal halves.
Ungrouped (odd n)Median = value at position (n+1)/2
Ungrouped (even n)Median = avg of n/2 and (n/2+1) positions
GroupedMedian = L + [(n/2 − cf)/f] × h

Where: L = lower class boundary of median class, cf = cumulative frequency before median class, f = frequency of median class, h = class width.

Grouped Median: n=50, median class is 30–40, L=30, cf=20, f=15, h=10
Median = 30 + [(25−20)/15]×10 = 30 + (5/15)×10 = 30 + 3.33 = 33.33

3.3 Mode

Mode: The value that appears most frequently in a dataset.
GroupedMode = L + [f₁−f₀ / 2f₁−f₀−f₂] × h

Where: L = lower limit of modal class, f₁ = frequency of modal class, f₀ = frequency of pre-modal class, f₂ = frequency of post-modal class, h = class width.

3.4 Relationship: Mean, Median & Mode

Unit III · Topic 4

Measures of Dispersion

Dispersion: The spread or variability of data around the central value. A smaller dispersion means data is concentrated near the mean; larger means data is spread out.

4.1 Range

RangeR = Maximum Value − Minimum Value
Coeff. of Range= (Max−Min)/(Max+Min)

4.2 Mean Deviation (MD)

Mean Deviation: Average of absolute deviations from the mean (or median/mode).
UngroupedMD = Σ|xᵢ − x̄| / n
GroupedMD = Σf|x − x̄| / Σf
Coefficient of MD= MD / Mean
Example (Ungrouped): Data: 4, 6, 8, 10, 12. Mean=8
|4−8|+|6−8|+|8−8|+|10−8|+|12−8| = 4+2+0+2+4 = 12
MD = 12/5 = 2.4

4.3 Standard Deviation (SD)

Standard Deviation (σ): The positive square root of the variance. Most widely used measure of dispersion.
Ungrouped (Direct)σ = √[Σ(xᵢ−x̄)²/n]
Ungrouped (Alternate)σ = √[(Σxᵢ²/n) − x̄²]
Grouped (Direct)σ = √[Σf(x−x̄)²/Σf]
Grouped (Step Dev.)σ = h × √[Σfu²/Σf − (Σfu/Σf)²]
Sample SDs = √[Σ(xᵢ−x̄)²/(n−1)]
Example: Data: 2, 4, 4, 4, 5, 5, 7, 9 (n=8), x̄=5
Σ(x−x̄)² = 9+1+1+1+0+0+4+16 = 32
σ = √(32/8) = √4 = 2

4.4 Variance

Varianceσ² = Σ(xᵢ−x̄)²/n   [= (SD)²]
Coefficient of VariationCV = (σ/x̄) × 100%

CV is used to compare variability of two datasets with different units or means. Lower CV = more consistent data.

MeasureFormulaMeritsDemerits
RangeMax − MinSimple, quickIgnores all middle values
Mean DeviationΣ|x−x̄|/nUses all valuesAbsolute values are algebraically inconvenient
Standard Deviation√[Σ(x−x̄)²/n]Mathematically rigorous, widely usedAffected by extreme values

1.1 Correlation — Definition

Correlation: A statistical technique that measures the degree and direction of the linear relationship between two variables X and Y.

If two variables tend to change together, they are said to be correlated.

1.2 Types of Correlation

Positive Correlation

Both variables move in the same direction. As X ↑, Y ↑.

e.g., Height and Weight

Negative Correlation

Variables move in opposite directions. As X ↑, Y ↓.

e.g., Price and Demand

Zero Correlation

No linear relationship between the variables.

e.g., Shoe size and IQ

Perfect Positive (r=+1)

All points lie on a straight line with positive slope.

Perfect Negative (r=−1)

All points lie on a straight line with negative slope.

Linear vs Non-linear

Linear: points near a straight line. Non-linear: points follow a curve.

1.3 Methods of Studying Correlation

  1. Scatter Diagram — Plot (X,Y) pairs; observe pattern visually
  2. Karl Pearson's Coefficient (r) — Quantitative measure for linear data
  3. Spearman's Rank Correlation (ρ) — For ordinal/ranked data
  4. Concurrent Deviation Method — Quick approximate method

1.4 Karl Pearson's Coefficient of Correlation

Directr = Σ(x−x̄)(y−ȳ) / √[Σ(x−x̄)² · Σ(y−ȳ)²]
Alternater = [nΣxy − ΣxΣy] / √{[nΣx²−(Σx)²][nΣy²−(Σy)²]}
Range−1 ≤ r ≤ +1
Value of rInterpretation
r = +1Perfect positive correlation
0.75 ≤ r < 1High positive correlation
0.5 ≤ r < 0.75Moderate positive correlation
0 < r < 0.5Low positive correlation
r = 0No linear correlation
−0.5 < r < 0Low negative correlation
−0.75 < r ≤ −0.5Moderate negative correlation
−1 < r ≤ −0.75High negative correlation
r = −1Perfect negative correlation
Example: n=5, Σx=15, Σy=25, Σx²=55, Σy²=135, Σxy=80
r = [5×80 − 15×25] / √{[5×55−225][5×135−625]}
= [400 − 375] / √{[275−225][675−625]}
= 25 / √{50×50} = 25/50 = 0.5 (Moderate positive)

1.5 Spearman's Rank Correlation Coefficient

Spearman's ρ (rho): Used when data is given in ranks or when the distribution is not normal. Suitable for ordinal data.
Formulaρ = 1 − [6ΣD²] / [n(n²−1)]
D = Rank of X − Rank of Y (difference in ranks)
Example: n=5, ranks given for X and Y:
D values: 1,−1,0,2,−2 → D²: 1,1,0,4,4 → ΣD²=10
ρ = 1 − [6×10]/[5×24] = 1 − 60/120 = 1 − 0.5 = 0.5
Tied Ranks: When two items share the same rank, assign average rank. Add correction factor: T = (t³−t)/12 per tie group, where t = number of tied items.
Unit IV · Topic 2

Regression Analysis

2.1 Regression — Definition

Regression: A statistical method used to establish the relationship between a dependent variable (Y) and one or more independent variables (X), for the purpose of prediction and estimation.

2.2 Uses of Regression Analysis

  • Predicting future values of Y given X
  • Identifying the strength and nature of the relationship
  • Estimating the impact of one variable on another
  • Useful in economics, business forecasting, agriculture, science

2.3 Correlation vs Regression

AspectCorrelationRegression
PurposeMeasures degree of relationshipPredicts value of one variable
DirectionSymmetric (X and Y are interchangeable)Asymmetric (X is predictor, Y is response)
NatureDescribes strength and directionDescribes nature of relationship (equation)
Coefficientr (dimensionless, −1 to +1)b (slope; has units)
LinesOnly one scatter plotTwo regression lines (one for X on Y, one for Y on X)
CausationDoes NOT imply causationMay suggest causal direction

2.4 Regression Lines

There are two regression lines:

Line of Y on X

Used to predict Y when X is given.

Equation: Y − ȳ = byx(X − x̄)

byx = r × (σy/σx)

Line of X on Y

Used to predict X when Y is given.

Equation: X − x̄ = bxy(Y − ȳ)

bxy = r × (σx/σy)

byx (Direct)byx = [nΣxy − ΣxΣy] / [nΣx² − (Σx)²]
bxy (Direct)bxy = [nΣxy − ΣxΣy] / [nΣy² − (Σy)²]
Relationshipr² = byx × bxy  ∴  r = ±√(byx × bxy)
Example: n=5, x̄=3, ȳ=4, σx=2, σy=3, r=0.8

byx = 0.8 × (3/2) = 1.2
Line of Y on X: Y − 4 = 1.2(X − 3) → Y = 1.2X + 0.4

bxy = 0.8 × (2/3) = 0.533
Line of X on Y: X − 3 = 0.533(Y − 4) → X = 0.533Y + 0.868

Verify: r = ±√(1.2 × 0.533) = ±√0.64 = ±0.8 ✓

2.5 Properties of Regression Lines

  1. Both regression lines pass through the point (x̄, ȳ) — the mean point.
  2. The angle between the two lines depends on r. When r=±1, lines coincide. When r=0, lines are perpendicular.
  3. The correlation coefficient r is the geometric mean of the two regression coefficients: r = ±√(byx·bxy)
  4. byx and bxy always have the same sign as r.
  5. If both regression coefficients are greater than 1 in absolute value, this is impossible. At least one must be ≤ 1.
